# 🧪 TheFlame App - Test Automation Framework

## 📋 Overview
Automated test suite for TheFlame dating app using **Playwright** + **Pytest** + **Allure**.

### 🎯 Features
- ✅ Page Object Model (POM) architecture
- ✅ Automated login & registration flows
- ✅ Referral & Redeem modal testing
- ✅ Allure reporting with screenshots
- ✅ One-click execution scripts
- ✅ Test data fixtures
- ✅ Parameterized tests

---

## 🚀 Quick Start

### Prerequisites
- Python 3.12+
- Allure 2.44.0 (installed via Scoop)

### 1. Environment Setup
```bash
# Activate virtual environment
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Install Playwright browsers
playwright install
